Overview

The International Flag of Earth - Image Generator overlays the "International Flag of Planet Earth" onto any image. The interface consists of a preview image, controls to modify the composite, and buttons to save the final image. The controls are grouped into three sections Background, Settings, and Options to indicate the type manipulation the control represents.

Use the "Update" button at the bottom of the interface to update the preview image after adjusting the controls. The image is generated with the device's processor (e.g. phone, computer, tablet, etc.) and may take some time with large images (2MB+). When the preview has finished updating, the browser will automatically scroll it into view.

Background

Use the file selection input to open the image selection dialog. Alternatively, tap or click the preview image to open the image selection dialog. Select an image from those available on your device, or other connected sources (e.g. camera, external media, cloud storage, etc.).

Once selected, the image will immediately load into memory and update the preview. This initial load may take some time for large images (2MB+). If no controls have been previously modified, a default composite configuration will be applied when the initial image is selected.

Settings

The controls in this section contain multiple selections or allow varying degrees of one parameter to be set. Adjust the controls to produce the desired composite image. Use the "Update" button at the bottom of the interface to update the preview image and view the adjustment.

  • Blend Mode

    Use this control to select the blend mode for the overlay. Each mode will produce a dramatically different effect when the background is blended with the overlay. Several standard image blend modes are implemented to provide a wide range of creative possibilities. More information about how image blend modes affect the composite can be found at: Wikipedia: Blend modes

  • Overlay Scale

    This control affects the relative size of the overlay. 100% will scale the overlay to the size of the largest possible square within the composite image (the lesser of height or width, aspect ratio preserved). A lower percentage will reduce the size of the overlay, while a larger percentage will scale the overlay beyond the borders of the composite. The blue matte will be extended or reduced to cover the background image when necessary.

  • Overlay Opacity

    Transparency can be achieved by setting the opacity of the overlay to a value less than 100%. Many of the blend modes alter the composite in intense ways which may be undesirable when applied with a fully opaque overlay. Use the opacity control to creatively tame this intensity... or apply deliberately.

Options

The controls in this section are binary or can only be applied exclusively. Enable or disable options with the radio buttons and checkboxes to produce the desired composite image. Use the "Update" button at the bottom of the interface to update the preview image and view the adjustment.

  • Fit Image

    To fit the overlay to the selected background image, depress this radio button. The dimensions of the selected background image will be used as the dimensions of the composite image. This also significantly reduces the effect of the "Smoothing" option, as the background image will not be scaled. The "900px · 600px" radio button will raise.

  • 900px · 600px

    To set the composite image width to 900 pixels and its height to 600 pixels, depress this radio button. The selected background image will be scaled to fit this size (aspect ratio preserved). These dimensions are grandfathered from the source image on Wikipedia (used to create this application) and reflect the 3 x 2 ratio of the original flag design. The "Fit Image" radio button will raise.

  • Smoothing

    Toggle this checkbox to enable or disable smoothing of the background image. With no smoothing applied, the square edges of the pixels will be preserved producing a generally sharper image. With smoothing applied, the background image will be interpolated as it's scaled to the to the dimensions of the composite. This reduces the appearance of sharp edges and smooths the transitions between colors in the background image.

  • Blue Matte

    Toggle this checkbox to enable or disable the blue background elements of the overlay. When disabled, only the flower will be drawn (in white) and may be creatively transformed with blend modes. Due to the computations involved in some of these modes, white may not modify the image when blended (e.g. Multiply, Darken).

Saving

After updating the preview image, the composite image may be saved to your device with the "Save" button. With relatively small images (e.g. avatars, profile pictures, etc.) this may be sufficient. Due to browser restrictions, large images (2MB+) may not be able to be saved. To mitigate this inconvenience, cloud storage providers can be used to receive the image from the browser as an upload.

  • Dropbox

    Tap or click the black Dropbox logo to connect the application to your Dropbox account. You will be prompted to login and grant "iFlag of Earth - Image Generator" permission to write files into an application folder ("Apps/B82 - iFlag of Earth") in your Dropbox. Once you have preformed this, Dropbox will return you to the application. The Dropbox logo will be full-color (indicating the connection), and tapping or clicking the "Save" button will now upload the composite image to Dropbox.

To return the "Save" button to its original function (save image to device), simply tap or click the connected (full-color) cloud storage provider logo to disable the connection.

Notes

Processing
All processing happens in-browser. Images are never uploaded to a server for processing. The time required to generate the composite image is dependent on the processing speed of the device.

Cloud Storage
At your discretion, the composite image may be uploaded to a cloud storage provider. The time required to upload the composite image is dependent on the speed of the device's Internet connection. Please be aware of any applicable charges that may be accrued during this data transfer.

Browser Compatibility

  • Firefox - all operations complete as expected
  • Safari - when saving to device, will open the composite image as a data url... use "Save As..." functionality to save image
  • Chrome - will not save large composite images to the device and will fail silently - no error or notification... use cloud storage
  • Internet Explorer - will not save large composite images to the device and will fail silently - no error or notification... use cloud storage
